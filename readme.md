# Базовый пакет Chunker

## Архитектура пакета

### Обзор структуры

Пакет `chunker2i/base` представляет собой базовый фреймворк для разработки с фокусом на SCSS-архитектуру и Laravel-интеграцию.

### Ключевые компоненты

#### 1. PHP-архитектура

**ServiceProvider (`AppServiceProvider.php`)**
- Расширяет стандартный `Illuminate\Support\ServiceProvider`
- Реализует паттерн "Package Provider" для Laravel
- Автоматически публикует SCSS-ресурсы в проект
- Подгружает view-шаблоны из пространств `utils`, `type`, `base`

**Traits (`ModifiersTrait.php`)**
- Реализует BEM-подход для CSS-классов
- Метод `buildModifiersClass()` генерирует модификаторы вида `.block_modifier`

#### 2. SCSS-архитектура

**Модульная структура с использованием `@use` и `@forward`**

```
resources/scss/
├── app.scss          # Точка входа
├── core/             # Ядро системы
│   ├── _colors.scss  # Продвинутая цветовая система
│   ├── _scale.scss   # Система масштабирования
│   ├── _tokens.scss  # Дизайн-токены
│   └── ...
├── config/           # Конфигурационные переменные
├── ui/               # UI-компоненты
├── utils/            # Утилиты и миксины
└── type/             # Типографика
```

#### 3. Архитектурные паттерны

**Цветовая система**
- Использует современное цветовое пространство OKLCH
- Автоматическая генерация оттенков и контрастных цветов
- Обратные карты для быстрого поиска цветов по значению
- Поддержка CSS-переменных

**Система масштабирования**
- Базовая единица 8px = 1rem
- Обратные карты для поиска переменных по значениям
- Автоматическая конвертация единиц

**Модульность**
- Чёткое разделение конфигурации и реализации
- Использование `@forward` для экспорта публичного API
- Инкапсуляция через `@use` с неймспейсами

#### 4. Интеграционные паттерны

**Laravel-интеграция**
- PSR-4 автозагрузка
- Автоматическая регистрация провайдера
- Публикация ресурсов в стандартные директории Laravel

**Паттерн публикации ресурсов**
- Метод `publishesScss()` для пакетных публикаций
- Гибкая система путей с хелперами
- Поддержка тегов для выборочной публикации

### Нюансы и особенности

1. **Современный SCSS**: Использование Sass modules вместо глобальных переменных
2. **Цветовая наука**: Расчёты в OKLCH с весовыми коэффициентами для контраста
3. **Производительность**: Обратные карты для O(1) поиска значений
4. **Гибкость**: Конфигурация отделена от реализации
5. **BEM-совместимость**: Встроенная поддержка модификаторов через trait

Пакет представляет собой хорошо структурированную основу для разработки с современными практиками CSS-архитектуры и Laravel-интеграцией.

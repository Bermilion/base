# Адаптивный дизайн

Используемый подход `mobile first`, таким образом интерфейс разрабатывается для мобильных устройств, а затем адаптируется под другие устройства.

Доступные медиазапросы:

| Класс | Размер в `rem` | Размер в `px` |
|:-----:|:--------------:|---------------|
|  .sm  | 80             | 640           |
|  .md  |	96             | 768           |
|  .lg  |	128            | 1024          |
|  .xl  |	160            | 1280          |
| .2xl  |	192            | 1536          |
